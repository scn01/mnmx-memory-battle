# MNMX Memory Battle

A stylish browser-based memory game where you compete against an adaptive AI opponent.

Flip cards, memorize positions, and outplay MNMX across multiple difficulty levels — from casual to impossible.

---

## Features

- AI opponent with simulated memory & strategy
- Multiple difficulty modes
- Smooth card flip animations
- Match history tracking with localStorage
- Fully responsive UI
- Pure HTML, CSS, and Vanilla JavaScript
- No frameworks or dependencies

---

## Preview

MNMX Memory Battle combines a modern dark UI with strategic gameplay inspired by classic concentration games.

### Difficulty Modes

| Difficulty | Board | AI Behaviour |
|---|---|---|
| Easy | 4×4 | Mostly random |
| Medium | 4×5 | Partial memory |
| Hard | 6×6 | Strong memory + strategy |
| Impossible | 6×6 | Perfect memory |
